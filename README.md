## 1、主要功能

* 通过继承**QAbstractNativeEventFilter**接口实现串口热插拔监测功能；
* 通过在QWidget中重写**nativeEvent**实现串口热插拔监测功能；
* 通过一个函数获取系统中所有可用串口名；
* 自动添加、移除可用串口；
* 鼠标、键盘热插拔监测。

## 2、适用系统

* Windows
